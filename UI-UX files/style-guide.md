# Design System

## Colors
- Primary: #1E3A8A (Deep Blue)
- Secondary: #F97316 (Vibrant Orange)
- Neutral: #6B7280 (Gray), #FFFFFF (White)

## Typography
- Headings: Poppins Bold
- Body: Inter Regular
- Accent: Italic for highlights

## Components
- Buttons: Rounded corners, hover color shift
- Cards: Event preview with image + title + date
- Navigation: Sticky top bar with logo + links
- Forms: Clean inputs with validation feedback
