# User Personas

## Student
- Goals: Discover events, register easily, track attendance
- Pain Points: Hard to find info, confusing registration
- Needs: Simple navigation, clear event details, mobile-friendly design

## Club Organizer
- Goals: Promote events, manage registrations, engage students
- Pain Points: Manual tracking, poor visibility
- Needs: Easy event creation, attendee management, attractive microsites

## Admin
- Goals: Oversee events, monitor participation
- Pain Points: Scattered data, limited control
- Needs: Dashboard with analytics, CRUD tools, secure authentication
